# 🚧 Road Accident Analysis Dashboard (Power BI + Python)

## 📌 Project Overview

This project analyzes road accident data to identify patterns, trends, and key risk factors contributing to accidents.
The dashboard helps stakeholders understand accident severity, yearly trends, and injury statistics for better decision-making.

The project combines **Python for data processing** and **Power BI for interactive visualization**.

---

## 🎯 Objectives

* Analyze road accident trends over multiple years
* Identify high-risk periods and severity patterns
* Visualize injury and fatality statistics
* Provide an interactive dashboard for insights

---

## 🛠️ Tools & Technologies Used

* **Python**
* **Pandas & NumPy**
* **Power BI Desktop**
* **CSV Dataset**
* **Data Visualization**

## 📊 Dashboard Features

* 📈 Year-wise accident trend
* 🚑 Injury vs fatality analysis
* 🗺️ State/region-wise distribution
* 📉 Comparative insights
* 🎛️ Interactive filters

## ▶️ How to Run the Project

### 🔹 Python Script

1. Install required libraries:

   ```bash
   pip install pandas numpy
   ```
2. Run:

   ```bash
   python road_accident.py
   ```

### 🔹 Power BI Dashboard

1. Download the `.pbix` file
2. Open in **Power BI Desktop**
3. Interact with filters and visuals
